🧩 Sunset Survival - 주요 시스템 소개

📜

퀘스트 시스템

계층형 구조로 구성된 퀘스트 시스템은 다양한 태스크 유형과 액션 타입을 통해 게임 진행을 유도합니다.

  • 계층형 구조
  • 진행도 저장
  • 퀘스트 저장소 연동
  • 태스크 유형: Kill, Interact, Pickup 등
⚒️

제작 시스템

레시피 기반 제작 시스템으로, 필요한 재료가 갖춰지면 실시간 검증을 통해 결과물을 생성합니다.

  • 재료 검증 및 소비
  • 실시간 UI 업데이트
  • 퀘스트 연동
  • 인벤토리 완전 연동
📦

인벤토리 시스템

모듈형 아키텍처로 구성된 인벤토리는 장비/아이템의 저장, 드래그 앤 드롭, 필터링 등을 제공합니다.

  • 드래그 앤 드롭 기능
  • 장비 시스템 연동
  • 아이템 내구도 관리
  • 다른 시스템과의 연동 (퀘스트/제작)
🌅

낮/밤 시스템

게임 내 시간 흐름을 기반으로 시각 효과와 사운드가 자동으로 전환되며, 밤에는 살인마가 활약합니다.

  • 24시간 시간 시스템
  • 조명 그라데이션 및 BGM 전환
  • SyncVar 기반 네트워크 동기화
  • 밤에 시야 제한 기능
